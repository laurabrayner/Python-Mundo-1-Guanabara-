# Print e Input

Esta pasta contém exercícios iniciais utilizando entrada e saída de dados em Python.

Os programas solicitam informações ao usuário e exibem mensagens na tela.

## Exercícios

### 1. Olá Mundo

Solicita o nome do usuário e exibe uma mensagem de boas-vindas.

### 2. Data de nascimento

Solicita o dia, mês e ano de nascimento do usuário e exibe a data formatada.

## Conceitos utilizados

* input()
* print()
* Variáveis
* Conversão de tipos (int)
* f-string
