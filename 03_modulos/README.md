# Uso de Módulos em Python

Esta pasta contém exercícios que utilizam módulos da biblioteca padrão do Python, como math e random.

Os programas realizam cálculos matemáticos, operações trigonométricas e manipulação de listas.

## Exercícios

* Raiz quadrada de um número
* Parte inteira de um número
* Cálculo da hipotenusa
* Funções trigonométricas
* Ordem aleatória de alunos

## Conceitos utilizados

* Importação de módulos
* Biblioteca math
* Biblioteca random
* Funções matemáticas
* Listas
* Embaralhamento de dados
