# Manipulação de Strings em Python

Esta pasta contém exercícios que trabalham com **manipulação de textos (strings)** em Python.

Os programas realizam verificações em palavras, busca de caracteres, separação de nomes e análise de frases digitadas pelo usuário.

---

## Exercícios

- Separando dígitos de um número
- Verificando as primeiras letras de um texto
- Procurando uma string dentro de outra
- Primeira e última ocorrência de uma string
- Primeiro e último nome de uma pessoa

---

## Conceitos utilizados

- Entrada de dados (`input`)
- Manipulação de strings
- Métodos de string (`strip`, `upper`, `split`, `count`, `find`, `rfind`)
- Indexação de strings
- Operadores de busca (`in`)
- Listas
- Tratamento de texto
