# Estruturas Condicionais em Python

Esta pasta contém exercícios que utilizam **estruturas condicionais** em Python para tomar decisões com base em condições.

Os programas verificam situações como comparação de números, cálculo de multas, identificação de números pares ou ímpares, cálculo de preços, reajuste salarial, verificação de ano bissexto e validação de formação de triângulos.

---

## Exercícios

- Jogo de adivinhação com número aleatório
- Verificação de multa por excesso de velocidade
- Verificação se um número é par ou ímpar
- Cálculo do preço de uma viagem por distância
- Cálculo de aumento salarial
- Verificação se três medidas formam um triângulo
- Verificação de ano bissexto

---

## Conceitos utilizados

- Estruturas condicionais (`if`, `else`)
- Operadores de comparação (`>`, `<`, `<=`, `==`, `!=`)
- Operadores lógicos (`and`, `or`)
- Operador módulo (`%`)
- Entrada de dados (`input`)
- Conversão de tipos (`int`, `float`)
- Biblioteca `random`
- Geração de números aleatórios
- Cálculos matemáticos
- Tomada de decisão em programas
